## Oversikt

Autolead er en fullstack webapplikasjon jeg har utviklet fra bunnen av. Plattformen demonstrerer hva som er mulig når man kombinerer moderne webutvikling med AI-teknologi.

## Teknisk omfang

Dette er det mest omfattende prosjektet i portfolioen min:

- **66,000+ linjer kode** skrevet fra scratch
- **290+ filer** i kodebasen
- **Fullstack arkitektur** med frontend, backend og database

## Tekniske highlights

### Frontend
- Next.js 14 med App Router
- Responsive UI med Tailwind CSS
- Real-time oppdateringer

### Backend
- API-routes med Next.js
- Databaseintegrasjon med Supabase
- Autentisering og brukeradministrasjon

### AI-integrasjon
- OpenAI GPT-4 for intelligent prosessering
- Naturlig språkforståelse
- Automatiserte workflows

## Hva dette viser

Dette prosjektet demonstrerer at jeg kan:
- Bygge komplekse applikasjoner fra scratch
- Integrere AI på en praktisk måte
- Håndtere hele stacken fra database til UI
